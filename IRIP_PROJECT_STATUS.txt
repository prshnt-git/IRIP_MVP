GITHUB ACTIONS — AUTOMATION
-----------------------------
daily-scrape.yml:
  - Cron: 2:30 AM UTC = 8:00 AM IST, every day
  - Steps: wake Render → run scripts/run_daily_scrape.py → POST CSV to /reviews/import-csv
  - Manual trigger: GitHub Actions → daily-scrape.yml → Run workflow (input: days_back)
  - Secret required: PIPELINE_SECRET_KEY (set in GitHub repo Settings → Secrets)

weekly-discovery.yml:
  - Cron: 3:00 AM UTC Monday = 8:30 AM IST Monday
  - Calls POST /products/discover?months_back=4
  - Manual trigger: available with months_back input
